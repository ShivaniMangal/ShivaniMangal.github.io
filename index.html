PRESENTATIONS

  • Demo of Automatic Water Level Controller

    • Water level is detected using HC-SR04 Ultrasonic Sensor, connected to a Raspberry Pi 3 Model B.
    • A permissible range is set through Java ME 8.3 code, to prevent overflow. When the range lies within threshold levels, an LED glows, the motor switches ON, and the tank is filled.
    • When the level exceeds the capacity limit, the LED and motor turn OFF, water flow to the tank stops.
